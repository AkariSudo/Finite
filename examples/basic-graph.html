--- layout: main title: 'Basic graph' lead: > A basic example of graph/workflow managed by Finite current: examples --- Overview --------

In this example, we'll see a basic Document workflow, following this graph : 

                              Reject
                       |-----------------|
  Transitions          |                 |
                       v    Propose      |       Accept
  States            Draft ----------> Proposed ----------> Accepted

  Properties     * Deletable                              * Printable
                 * Editable
Implement your document class ----------------------------- {% highlight php %} state; } public function setFiniteState($state) { $this->state = $state; } } {% endhighlight %} Configure your graph -------------------- {% highlight php %} 'Document', 'states' => [ 'draft' => [ 'type' => Finite\State\StateInterface::TYPE_INITIAL, 'properties' => ['deletable' => true, 'editable' => true], ], 'proposed' => [ 'type' => Finite\State\StateInterface::TYPE_NORMAL, 'properties' => [], ], 'accepted' => [ 'type' => Finite\State\StateInterface::TYPE_FINAL, 'properties' => ['printable' => true], ] ], 'transitions' => [ 'propose' => ['from' => ['draft'], 'to' => 'proposed'], 'accept' => ['from' => ['proposed'], 'to' => 'accepted'], 'reject' => ['from' => ['proposed'], 'to' => 'draft'], ], ]); $document = new Document; $stateMachine = new Finite\StateMachine\StateMachine($document); $loader->load($stateMachine); $stateMachine->initialize(); {% endhighlight %} At this point, your Workflow / State graph is fully accessible to the state machine, and you can start to work with your workflow. Working with workflow --------------------- ### Current state {% highlight php %} getCurrentState()->getName(); // string(5) "draft" // List the currently accessible properties, and their values $stateMachine->getCurrentState()->getProperties(); // array(2) { // 'deletable' => bool(true) // 'editable' => bool(true) // } // Checks if "deletable" property is defined $stateMachine->getCurrentState()->has('deletable'); // bool(true) // Checks if "printable" property is defined $stateMachine->getCurrentState()->has('printable'); // bool(false) {% endhighlight %} ### Available transitions {% highlight php %} getCurrentState()->getTransitions()); // array(1) { // [0] => string(7) "propose" // } // Check if we can apply the "propose" transition var_dump($stateMachine->getCurrentState()->can('propose')); // bool(true) // Check if we can apply the "accept" transition var_dump($stateMachine->getCurrentState()->can('accept')); // bool(false) {% endhighlight %} ### Apply transition {% highlight php %} apply('accept'); } catch (\Finite\Exception\StateException $e) { echo $e->getMessage(); } // The "accept" transition can not be applied to the "draft" state. // Applying a transition $stateMachine->apply('propose'); $stateMachine->getCurrentState()->getName(); // string(7) "proposed" $document->getFiniteState(); // string(7) "proposed" {% endhighlight %}
